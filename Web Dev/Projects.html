Project 1

Remote Controlled Land Mine Detector Using Zigbee Protocol

Period:  January 2011 - April 2011
Team Size:  4
Role:  Team Lead, Circuit designer and prototyping
Skills Used:  Embedded C, Electronic Circuit Designing, PCB board Soldering
Tools Used:  Zigbee XBee Module S2C 802.15.4, PIC 16F877A
Automation Testing tool:  MP Lab IDE, Proteus Circuit Simulation.
Project Type:  Embedded System
Place:  Bharath University

Project Summary

  • This project is mainly applicable in the military application involves a rover, which will be traveling along different paths according to the command given by the user through Zigbee transceiver.
  • The metal detecting sensor of the rover senses the metals and gives the warning signal to the PC if the type of metals used in explosives.
  • The location (latitude and longitude)where the land mine is buried is captured by the GPS receiver and sent to the user's RCU(Remote Control Unit).

Project 2

Simulation of Generator Unit in Thermal Power Station Using LabVIEW Software

Period: February 2013 - April 2013
Project Type:   Graphical User Interfaces for Embedded Systems
Role:   Embedded GUI designer
Technologies:   LabVIEW
Skills Used:   GUI Embedded GUI Programming
Team Size:   Single
Place:   Sathyabama University

Project Summary

  • Laboratory Virtual Instrumentation Engineering Workbench (LabVIEW) are widely used in industry for supervisory control and data acquisition of industrial processes.
  • This paper describes the simulation of a system for laboratory-based thermal power plant generator setup using LabVIEW data logging and supervisory control (DSC) module.
  • Using the input, output, and transfer function parameters, the simulation of the generator unit and alarm handling technique is achieved.
  • This simulation results can be used for efficient use of coal burn to get higher throughput of power output from the generator in the thermal power plant.
